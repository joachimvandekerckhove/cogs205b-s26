## Syllabus

### Course description

This graduate course develops core **computational lab skills** and collaborative software habits for cognitive and neural science research. The opening modules focus on reproducible environments and version-control practice: containerization with Docker, terminal workflow, SSH authentication, Git command fluency, pull-request discipline, and project file organization. The emphasis is practical: you should be able to set up a reproducible workflow, move code safely between local work and shared repositories, and maintain a structure that other researchers can read and reproduce.

**Textbook:** Russell A. Poldrack, *Better Code, Better Science* — [bettercodebetterscience.github.io/book](https://bettercodebetterscience.github.io/book/).

### Prerequisites

Prior programming experience (we will use Python) and general computer literacy. No prior Docker, Git, or LaTeX background is assumed.

### Assignments and grading

Each assignment describes what to submit, when it is due, how it is scored, and the grading criteria. Follow those instructions for the week you are in.
