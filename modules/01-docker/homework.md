# Homework

Four components of the homework:

- Write a Python function \texttt{root} that computes the square root of an input manually, without using a sqrt function (25pt).
- Write a second Python function \texttt{test\_root} that returns TRUE if your \texttt{root} function works exactly as expected and FALSE otherwise (25pt).
- This must run in the container (25pt).
- Do not ask clarification questions (25pt).
