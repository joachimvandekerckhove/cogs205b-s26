# Homework (Module 01)

Four components of the homework:

- Write a Python function `root` that computes the square root of an input manually, without using a sqrt function (25pt).
- Write a second Python function `test_root` that returns `TRUE` if your `root` function works exactly as expected and `FALSE` otherwise (25pt).
- This must run in the class container (25pt).
- Do not ask clarification questions (25pt).
